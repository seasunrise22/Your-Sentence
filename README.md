# Your-Sentence
- 개발인원 : 1명
- 역할
  - 전체
## Introduction
원하는 문구를 작성하여 스마트폰 잠금화면에 항상 띄워둘 수 있는 안드로이드 애플리케이션입니다.

각종 옵션들을 스스로 선택할 수 있도록 구성하여 다채롭고 편리하게 자기만의 잠금화면을 꾸밀 수 있도록 구현하였습니다.

## Development Environment
- IDE : Android Studio
- Language : Java
